### How to verify

1. **Check the file hash.** Run `sha256sum` on the downloaded APK and confirm it matches the **File SHA‑256** above.

   ```bash
   sha256sum SecurQbit-Crypto-1.18.9-30.apk
   ```

2. **Confirm the signer.** With the Android build tools installed, run:

   ```bash
   apksigner verify --print-certs SecurQbit-Crypto-1.18.9-30.apk
   ```

3. **Compare the certificate digest.** The certificate digest it prints must equal the **Signing certificate SHA‑256** above. If either value differs, **do not install.**
